# Animesh Acharya — Portfolio

Personal portfolio website of Animesh Acharya — Front-End Developer & Data Visualization Specialist based in Kathmandu, Nepal.

**Live site:** deployed on Vercel

## Tech

- React 18 + Vite
- Pure CSS (no framework) — custom teal/gold theme, Cinzel + Jost typography
- CSS 3D spiral-staircase skills carousel, ambient orb + golden dust background

## Develop

```bash
npm install
npm run dev     # local dev server
npm run build   # production build to dist/
```

All site content (bio, experience, projects, skills, education, references) lives in `src/data.js`.
